# Student Placement Predictor

Predicts whether a student will get placed based on academic performance and skills.

## Results
| Model | Accuracy |
|-------|----------|
| Logistic Regression | 88.37% ✅ Best |
| Decision Tree | 83.72% |
| KNN | 79.07% |

## What I did
- Cleaned and preprocessed 215 student records
- Encoded categorical variables using LabelEncoder
- Compared 3 ML models and selected the best performer
- Created visualizations to analyze placement trends

## Visualizations
- Placed vs Not Placed distribution
- 10th marks vs Placement outcome
- Model accuracy comparison chart

## Tech Stack
Python · pandas · NumPy · scikit-learn · Matplotlib · Seaborn

## Dataset
Campus Recruitment dataset from Kaggle
