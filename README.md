# Student Placement Predictor

Predicts whether a student will get placed based on academic and skill data.

## Models Used
- Logistic Regression (Best — 88.37% accuracy)
- Decision Tree (83.72%)
- KNN (K-Nearest Neighbors) (79.07%)

## Dataset
- 215 student records
- Features: CGPA, internships, projects, communication skills etc.

## Libraries
Python, pandas, NumPy, scikit-learn

## Results
Logistic Regression outperformed other models with 88.37% accuracy on test data.udent-placement-predictor
